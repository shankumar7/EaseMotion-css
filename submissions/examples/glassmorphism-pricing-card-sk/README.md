# Glassmorphism Pricing Card

A modern pricing tier card utilizing the "Frosted Glass" aesthetic.

## Files
- `demo.html`: The HTML structure for the background and the glass card.
- `style.css`: The CSS that generates the backdrop blur and semi-transparent borders.
- `README.md`: This file.

## Features
- **Backdrop Blur:** Uses `backdrop-filter: blur(10px)` to create the frosted glass effect, which dynamically blurs the background behind it.
- **Translucent Borders:** Uses `rgba(255, 255, 255, 0.2)` on borders to simulate the reflective edge of glass.
- **Hover Lift:** Adds a subtle shadow increase and upward transform on hover to make the card feel interactive.

## Usage
Wrap your content in a `.glass-pricing-card`. For the effect to be visible, the card MUST be placed over a colorful, complex, or patterned background.
