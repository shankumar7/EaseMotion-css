# Smart Stack Utilities

Smart Stack Utilities provide a simple way to create vertically spaced layouts using a single utility class.

## Available Classes

- `.ease-stack-sm`
- `.ease-stack`
- `.ease-stack-lg`
- `.ease-stack-xl`

## Example

```html
<div class="ease-stack">
  <h2>Title</h2>
  <p>Description</p>
  <button>Action</button>
</div>
